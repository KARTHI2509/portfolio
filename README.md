# Portfolio - Karthik Thalipineni

My personal portfolio website built with HTML, CSS, and JavaScript.

## Features
- **Modern Responsive Design**: Balanced layouts for desktop, tablet, and mobile devices.
- **3D Interactive Cards**: Mouse-tracking 3D parallax tilt effects on the profile card.
- **Custom Trailing Cursor**: Dual-element smooth cursor tracking with micro-interactions on hover.
- **Dynamic Particles Background**: Interactive network nodes using Particles.js.
- **Filterable Projects Grid**: Smooth filtering transitions for project categories.
- **Dark/Light Mode**: Elegant corporate blue light theme and slate-900 dark theme.
